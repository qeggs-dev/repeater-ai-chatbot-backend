## 变量模板展开
使用大括号包裹的变量会被自动展开
使用空格来分隔变量名和参数
比如\{copytext "text" 10 " "}就可以生成10个"text"
Sample Output:
```
{copytext "text" 10 " "}
```
使用\{text_matrix "text" 5 3} 可以生成一个5列3行的文本矩阵
Sample Output:
```
{text_matrix "text" 5 10}
```
这些就会被替换到输出文本中
